# МЕСТО - Сервис для добавления фотографий
Проект №4

## О проекте

Проект создан с помощью применения адаптивной верстки под разные устройства.
Реализована часть логики на JavaScript. Есть возможность редактировать данные профиля.

## Технологии

+ HTML/CSS/JS
+ Методология БЭМ
+ Flexbox
+ Grid
